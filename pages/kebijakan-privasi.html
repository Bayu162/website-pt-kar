Kebijakan Privasi

Halaman ini menjelaskan bagaimana PT Ketiara Aliran Rezeki (PT KAR) memperlakukan data pengunjung website ini. Terakhir diperbarui: 2026.

Website Statis Tanpa Database

Website ini bersifat statis dan tidak memiliki database, sistem login, maupun panel admin. Kami tidak menyimpan data pengunjung di server perusahaan.

Data yang Dapat Anda Berikan

Jika Anda menghubungi kami melalui website ini, data yang Anda kirimkan (nama, pesan, email) akan diteruskan langsung melalui salah satu kanal berikut, dan tidak melewati server maupun database kami:

  • Tombol/formulir WhatsApp — pesan dikirim langsung ke aplikasi WhatsApp resmi PT KAR.
  • Link email (mailto) — pesan dikirim langsung melalui aplikasi email Anda ke alamat resmi PT KAR.
  • Google Maps — hanya menampilkan peta lokasi kantor, tanpa mengumpulkan data pribadi Anda.

Penggunaan Layanan Pihak Ketiga

Apabila di kemudian hari website ini menggunakan layanan formulir pihak ketiga (misalnya Google Form, Formspree, atau layanan sejenis), data yang Anda kirimkan akan diproses sesuai kebijakan privasi masing-masing penyedia layanan, dan hal ini akan diinformasikan sebelum formulir tersebut digunakan.

Cookie dan Analitik

Website ini tidak menggunakan cookie pelacakan. Apabila di kemudian hari Google Analytics atau alat serupa diaktifkan, halaman ini akan diperbarui untuk menjelaskan data yang dikumpulkan.

Kontak

Pertanyaan seputar kebijakan privasi ini dapat disampaikan melalui info@ketiara.id atau halaman Kontak.
