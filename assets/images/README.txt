Folder ini untuk menyimpan foto dan dokumentasi asli perusahaan
(hero banner, foto kegiatan, foto peserta, dsb.) sesuai Brief bagian 12.

Panduan ukuran file (Brief bagian 17):
- Hero banner   : maks. sekitar 300-500 KB
- Gambar konten : maks. sekitar 150-300 KB
- Thumbnail     : maks. sekitar 100-200 KB

Gunakan format WebP jika memungkinkan, dan kompres gambar sebelum diunggah.
